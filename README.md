## weekend-meme-stream-bot

### AKA. *pod*
Discord bot for our weekend meme stream  
bot testing on channel #bot-test

<br>

### instruction

* run `npm i` after cloning
* if you have nodemon: `nodemon`
* otherwise,  `npm start`

<br>

### current features

* swearing
* basic messages
* restart & stop commands
* play music:
  + play tracks
  + show music queue
